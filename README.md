# 📊 AdventureWorks Sales Analysis — Power BI Dashboard

An end-to-end **sales analytics dashboard** built in **Power BI** using the AdventureWorks dataset.  
This project provides deep insights into sales performance, customer behavior, and product trends to support data-driven business decisions.

---

## 📌 Problem Statement

Businesses often struggle to track sales performance, understand customer purchasing behavior, and identify profitable products.

This project aims to:

- Analyze sales trends across time and regions  
- Identify top-performing products and categories  
- Monitor customer purchasing patterns  
- Support strategic decision-making using data insights  

---

## 🛠 Tools & Technologies

- **Power BI** — Data modeling & dashboard development  
- **DAX** — KPI calculations and measures  
- **Power Query (M)** — Data cleaning & transformation  
- **Excel / CSV** — Data source (AdventureWorks dataset)  

---

## 📂 Project Overview

The dashboard analyzes sales data using key business dimensions:

- Total Sales, Profit, and Orders  
- Sales trends over time (monthly/yearly)  
- Product category and sub-category performance  
- Customer segmentation and purchasing behavior  
- Regional sales distribution  
- Top-selling products analysis

## 📈 Key KPIs Tracked

- Total Sales Revenue  
- Total Profit  
- Total Orders  
- Profit Margin (%)  
- Sales Growth Trend  
- Top Products by Revenue  
- Customer Count  

---

## 🔍 Key Insights

- Certain product categories contribute the majority of revenue  
- A small group of products generates high profit  
- Sales show clear seasonal trends  
- Some regions outperform others in revenue generation  
- Repeat customers contribute significantly to total sales  

---

## 📊 Visualizations Used

- **Line Charts** — Sales trends over time  
- **Bar Charts** — Product and category performance  
- **Donut Charts** — Sales distribution  
- **Maps** — Regional sales analysis  
- **Cards** — KPI monitoring  
- **Slicers** — Region, Category, Time filters
